# Trust Account Integrity Engine — Demo Frontend

React + TypeScript + Vite frontend for the
[Trust Account Integrity Engine](https://github.com/dangminhcuong07-cpu/trust-account-integrity-engine-core)
— a deterministic compliance checker for New Zealand law firm trust accounts.

**Live demo →** https://trust-account-integrity-engine.vercel.app/

## What this repo is

The presentation layer only. It renders pre-computed rule results from
synthetic data: an exception report view with per-violation evidence and
regulation citations, an interactive sandbox, and a downloadable sample
report. Fully static — no backend, no uploads, no client data.

The rules engine, 333-test suite, and verified regulation mapping live in
the [core repository](https://github.com/dangminhcuong07-cpu/trust-account-integrity-engine-core),
which is the main artifact of this project.

## Stack & decisions

- **React 18 + TypeScript + Vite**, deployed on Vercel (CI on push)
- **Static-by-design:** the production product runs on-premise inside a
  law firm; this public demo therefore ships only synthetic, pre-computed
  results so no data path to a server exists at all
- Typed payload contract (`data.ts`) generated by the engine's pipeline,
  keeping frontend display strings in lockstep with the engine's
  regulation citations

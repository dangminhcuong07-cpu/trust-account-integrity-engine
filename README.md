# Trust Account Integrity Engine — Demo Frontend
React/Vite frontend for the [Trust Account Integrity Engine](LINK-TO-ENGINE-REPO)
— a deterministic compliance checker for NZ law firm trust accounts.
Live demo: https://trust-account-integrity-engine.vercel.app/
Renders pre-computed results from synthetic data. No backend; nothing is uploaded.
The rules engine, tests, and regulation mapping live in the main repository.
